# Vue 3 + TWD Example

This example should help get you started developing with Vue 3 and TWD in Vite.

## Running the example

```bash
npm install
npm run dev
```

To execute this example, you need to build the TWD library first. in the root of the project, run the following command:

```bash
npm run build
```

This will build the TWD library and then you need to copy the files to the `examples/vue-twd-example/src/dist` directory.

Then, you can run the example by running the following command:

```bash
npm run dev
```
